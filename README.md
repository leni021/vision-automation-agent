# Autonomous UI Automation & Vision Agent

## Technical Overview

Motor de automatizacion avanzado disenado para la navegacion e interaccion en Single Page Applications (SPAs) de alta complejidad. El sistema integra Playwright con modelos de vision artificial (GPT-4o) para resolver flujos de interfaz dinamicos que superan las capacidades de los selectores estaticos tradicionales.

## Key Engineering Features

- Computer Vision Logic: Interpretacion de contextos visuales para la toma de decisiones en tiempo real sobre elementos no estructurados del DOM.
- State Persistence Architecture: Implementacion de flujos de autenticacion hibridos con persistencia de estado y manejo de sesiones seguras.
- DOM Resilience: Estrategias de interaccion forzada y bypass de capas de interfaz para garantizar la ejecucion en entornos web reactivos.

## Stack

- Backend: Node.js
- Automation: Playwright
- AI: GitHub Models API (GPT-4o)
