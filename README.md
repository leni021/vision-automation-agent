# Autonomous UI Automation & Vision Agent

## Technical Overview

Motor de automatización avanzado diseñado para la navegación e interacción en Single Page Applications (SPAs) de alta complejidad. El sistema integra Playwright con modelos de visión artificial (GPT-4o) para resolver flujos de interfaz dinámicos que superan las capacidades de los selectores estáticos tradicionales.

## Key Engineering Features

- Computer Vision Logic: Interpretación de contextos visuales para la toma de decisiones en tiempo real sobre elementos no estructurados del DOM.
- State Persistence Architecture: Implementación de flujos de autenticación híbridos con persistencia de estado y manejo de sesiones seguras.
- DOM Resilience: Estrategias de interacción forzada y bypass de capas de interfaz para garantizar la ejecución en entornos web reactivos.

## Stack

- Backend: Node.js
- Automation: Playwright
- AI: GitHub Models API (GPT-4o)
